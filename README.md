# apple-stock-price-prediction-next-30day-s-
Apple Stock Price Prediction (Next 30 Days)   This project forecasts Apple stock prices using historical data (2012–2019). Models applied: ARIMA, SARIMA, and XGBoost. After evaluation, XGBoost achieved the lowest error (RMSE ≈ 33.77, MAE ≈ 23.23). The model is deployed via Streamlit for interactive real‑time forecasting.
